# LiftLab

Strona internetowa firmy oferującej profesjonalne plany treningowe oparte na nauce. Trening siłowy z wiedzą naukową — skuteczniej, bezpieczniej i z realnymi efektami.

## Technologie

- **React 19** + **TypeScript**
- **Vite** — narzędzie budowania
- **Tailwind CSS** — stylowanie
- **React Router** — routing
- **Framer Motion** — animacje
- **Lucide React** — ikony

## Struktura projektu

```
src/
├── components/       # Komponenty stron
│   ├── Hero.tsx      # Sekcja powitalna
│   ├── About.tsx     # O nas
│   ├── Testimonials.tsx
│   ├── Product.tsx
│   ├── Layout.tsx    # Układ z Header i Footer
│   └── HomePage.tsx
├── UI/
│   ├── Header.tsx    # Nawigacja
│   └── Footer.tsx    # Stopka
├── App.tsx
├── main.tsx
└── index.css
```
